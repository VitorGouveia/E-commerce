Overview

This is a scalable, easy to use, "batteries included" boilerplate for your e-commerce projects, by using a specific set of technolgies the learning curve is almost flat, write complex SQL queries in no time, feel the power of a simple but yet great authentication system and build your e-commerce now! ðŸš€

Work with complex data and relationships

Make SQL joins, queries, searchs all in a matter of seconds! with prisma everything is possible. We already include multiple types of relationships, such as One-to-Many, One-to-One and Many-to-Many.

Make everything faster with cluster mode

A node.js native cluster mode is already implemented, just change your server.ts Cluster(false) function to true and see the magic happen. More details about node.js cluster mode here

Installation

To get started just clone the repository:

git clone https://github.com/EsquemaFlorescer/E-commerce.git

Then, access the project folder:

cd E-commerce/

Finally, install all the dependencies:

yarn
# or
npm i

Getting Started

Now you will learn how to use our api routes and start your e-commerce, remember that we only provide the back-end, you need to build your own UI of your app

Now let's create a user, first call the user creation endpoint

POSThttp://localhost:3333/user

Then, supply some information in JSON

{ "name": "vitor", "email": "vitor@vitor.com", "cpf": "000.000.000-00", "password": "12345" }

Finally, receive your response, also in JSON
if you want more details on how the user is created, click here

{ "auth": true "access_token": JWT TOKEN "user": { "id": UUID V4, "created_at": TIMESTAMP, "name": "vitor" "last_name": null, "cpf": "000.000.000-00", "email": "vitor@vitor.com", "password": ENCRYPTED PASSWORD } "message": "User created with success!" }
